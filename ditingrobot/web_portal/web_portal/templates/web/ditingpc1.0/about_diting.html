关于我们

谛听机器人（www.ditingai.com）作为一家中文语义与认知计算的产品服务提供商，北京谛听机器人科技有限公司致力于自然语言理解技术（NLU）的持续创新与应用，重视人机交互的流畅化、自然化，践行让机器人更加理解人类的使命，顺应国内语义理解市场的发展应运而生。


通过十四年的摸索、实验和探究，谛听机器人研发出了基于语言学的句法分析算法——深度语义理解技术DSA，能有效解决人机交互自然流畅的根本问题。


2016年6月1日，谛听机器人语义理解开放平台正式上线，开始服务于更多的企业和终端用户。平台的推出大大降低了开发者使用语义技术的门槛，开发者可以利用这项技术，在交互方式上进行新的尝试和突破，使得人机交互更加简单与多样。谛听机器人可以接入微信公众号或网站，还可以通过平台提供的机器人API开发出各种不同的机器人产品，让每一个企业和个人都能轻松创建专属于自己的机器人。
